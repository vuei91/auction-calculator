# 부동산 경매 수익 계산기

React + Vite로 만든 부동산 경매 수익 계산기입니다.

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

## 기능

- 낙찰금액 및 판매예상금액 입력
- 취득세 자동 계산 (주택 수, 지역별)
- 경락잔금대출이자 자동 계산
- 종합부동산세 자동 계산
- 양도소득세 자동 계산 (보유기간, 주택유형별)
- 중개수수료 자동 계산 (서울시 기준)
- 총 투자금액, 순수익, 수익률 계산

## 기술 스택

- React 18
- Vite
